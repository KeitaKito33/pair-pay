■Optional.ofNullable(user)
userを基本的には返すが、もしuserがnullだった場合に、nullではなく空を返す。
以下のように「もし空だったらエラーを返す」みたいなエラー文を書きやすくなる。

userRepository.findById(userId)
        .orElseThrow(() -> new RuntimeException("..."));
